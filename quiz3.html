Trees and Hashing Quiz

1. What is a binary tree in data structures?
A. A tree structure with at most two children for each node.
B. A tree structure with unlimited children for each node.
C. A tree structure with no children for each node.
D. None of the above.
2. What is the process of visiting all the nodes of a tree and performing an operation at each node exactly once called?
A. Insertion
B. Traversal
C. Deletion
D. Rotation
3. In a hash table, what is a collision?
A. When two different keys hash to the same index.
B. When a key hashes to a unique index.
C. When a key is successfully inserted into the hash table.
D. None of the above.
4. What is the purpose of a hash function in hashing?
A. To perform encryption.
B. To generate random numbers.
C. To map keys to indices in the hash table.
D. None of the above.
5. In a binary search tree, what is the property of the left subtree of a node?
A. All nodes have greater values than the node.
B. All nodes have smaller values than the node.
C. It can have nodes with either greater or smaller values than the node.
D. None of the above.
6. What is the primary advantage of using a self-balancing binary search tree (e.g., AVL tree)?
A. Faster insertion and deletion operations.
B. Smaller memory footprint.
C. Improved search performance.
D. Better aesthetics.
7. What is the load factor in the context of hashing?
A. The weight of the hash function.
B. The ratio of the number of elements to the size of the hash table.
C. The speed of the hash table.
D. None of the above.
8. What is the purpose of a hash collision resolution technique?
A. To avoid collisions entirely.
B. To efficiently handle collisions and store multiple keys at the same index.
C. To increase the size of the hash table.
D. None of the above.
9. What is the time complexity of searching in a balanced binary search tree?
A. O(1)
B. O(log n)
C. O(n)
D. O(n^2)
10. In a hash table, what is the purpose of the array used for storage?
A. To store keys directly.
B. To store values directly.
C. To store indices generated by the hash function.
D. To store linked lists or other data structures for handling collisions.
